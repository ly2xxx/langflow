Here's an example of how you can define a Python function tool in LangFlow: ```python from langflow import PythonCodeStructuredTool, PythonREPLToolComponent class AddTwoNumbers(PythonCodeStructuredTool): def __init__(self): super().__init__() self.name = "Add Two Numbers" self.description = "Adds two numbers together." self.code = """ def add_two_numbers(a, b): return a + b """ self.fields = [ {"name": "a", "type": "number"}, {"name": "b", "type": "number"} ] class Tool(PythonREPLToolComponent): def __init__(self): super().__init__() self.add_two_numbers = AddTwoNumbers() tool = Tool() ``` This code defines a tool that adds two numbers together. The `AddTwoNumbers` class is a subclass of `PythonCodeStructuredTool`, which provides the basic structure for defining a Python function tool. The `fields` attribute specifies the input parameters for the function, and the `code` attribute contains the source code for the function. You can then use this tool in your LangFlow flow by adding it to the `tools` list: ```python flow = Flow() flow.add_tool(tool) ``` This will create a new flow that includes the "Add Two Numbers" tool. You can then execute the flow and pass input values for the function's parameters, like so: ```python result = flow.execute({"a": 2, "b": 3}) print(result) # Output: 5 ``` This will add 2 and 3 together and print the result.
